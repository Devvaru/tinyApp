## Features
* Generate short URLs
* Edit URLs
* Delete URLs
* Account creation
* URL storage

## Dependencies

* bcryptjs
* cookie-session
* ejs
* express

## Getting Started

* Install all dependencies (using the `npm install` command).
* Run the development web server using the `node express_server.js` command.
* Use `ctrl/cmd+c` to stop the server
